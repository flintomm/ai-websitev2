3) Trigger: [recurring check-in/self-service/custom]
4) Payload: [scripts/packages/config profiles]

Smart Group Logic Template

Reference format for targeting conditions and testing strategy.

Template: Smart Group Criteria
- Platform: macOS version >= [x]
- Management: Jamf binary present
- Security: FileVault = Enabled
- Exceptions: Department != [excluded group]
- Dry run group: [5-10 pilot devices]
