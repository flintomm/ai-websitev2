# Budget Calculator Dashboard

Spreadsheet-style budget planner for tracking planned vs actual spending across a custom period.

## What it does

- Select period: weekly, monthly, yearly
- Set a budget target
- Add/edit/remove rows with category, item, planned, and actual amounts
- Live center summary that shows:
  - under budget
  - over budget
  - on target
- Persists data for the current browser session (`sessionStorage`)
